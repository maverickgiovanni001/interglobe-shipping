InterGlobe.cloud — Single-file PHP app
======================================
DEPLOY (drag & drop):
  1. Upload index.php to your web root (public_html, htdocs, www).
  2. Open https://yourdomain.com/index.php  → done.
  3. (Optional) Upload .htaccess for pretty URLs (/book, /track, /about).

REQUIREMENTS: PHP 7.4+ with write permission to create ./data/ folder.
